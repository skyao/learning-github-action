## 概览

在工作流程中使用的操作可以定义于：

- 与你的工作流文件相同的存储库
- 任何公共存储库
- Docker Hub 上发布的 Docker 容器镜像

GitHub Marketplace 是您查找 GitHub 社区创建的操作的中心位置。 [GitHub Marketplace 页面](https://github.com/marketplace/actions/)可用于按类别筛选操作。
